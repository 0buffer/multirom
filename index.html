                Boot manager's main screen

MultiROM allows you to boot ROM from SD card while the main ROM (the one in internal memory) stays untouched. It also features simple boot manager, which shows up during boot and allows you to chose which ROM to boot.

Boot manager can also replace LG's charging logo, which would normally show up, if you would turn on the phone by plugging-in the USB cable. You can then see percentage of battery charging level or chose to boot android WITHOUT restart.

It also features good old tetris game.

See thread on XDA for more info: http://forum.xda-developers.com/showthread.php?t=1304656
